Learn
========

Requirements
------------------------

* Install docker latest from https://www.docker.com.

### Notes

Start container:
```
docker compose up -d
```

Stop container:
```
docker compose stop
```

Setup
------------------------

- Open the hosts file `sudo nano /etc/hosts` and add `127.0.0.1 learn.test`.
- Start container
- Please check http://learn.test

### For Windows

- Switch to *nix based OS.
- Follow instructions above.
